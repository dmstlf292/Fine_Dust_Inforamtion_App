# Fine_Dust_Inforamtion_App
Fine Dust Inforamtion App
* 자동 현재위치에 대하여 미세먼지 정보 알려주기
* 추가하고 싶은 위치 생성하기
* 전체지역 삭제 혹은 지우고 싶은 지역 삭제하기
* 공공데이터 미세먼지 api 사용으로 실시간 반영
